# amazon_clone
An Amazon clone built to simulate core e-commerce functionalities, including product browsing, user authentication, shopping cart management, and order placement.
# Features
<b>1.User Authentication</b>: Secure sin-up and login system.
</br>
<b>2.Search Functionality</b>: Quick and efficient product searching. 
  </br>
<b>3.Product Details Page</b>: Detailed product descriptions,images.
  </br>
<b>4.Admin Panel</b>: An interface for managing products effectively.
  </br>
<b>5.Responsive Design</b>: Optimized for both desktop and mobile users.
# Technologies Used
Frontend: HTML, CSS
