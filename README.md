# amazon_clone
An Amazon clone built to simulate core e-commerce functionalities, including product browsing, user authentication, shopping cart management, and order placement.
# Features
User Authentication : Secure sin-up and login system.
Search Functionality: Quick and efficient product searching. 
Product Details Page: Detailed product descriptions,images.
Admin Panel         : An interface for managing products effectively.
Responsive Design   : Optimized for both desktop and mobile users.
# Technologies Used
Frontend: HTML, CSS
