# amazon_clone
An Amazon clone built to simulate core e-commerce functionalities, including product browsing, user authentication, shopping cart management, and order placement.
# Features
<b>.User Authentication</b>: Secure sin-up and login system.
</br>
<b>.Search Functionality</b>: Quick and efficient product searching. 
  </br>
<b>.Product Details Page</b>: Detailed product descriptions,images.
  </br>
<b>.Admin Panel</b>: An interface for managing products effectively.
  </br>
<b>.Responsive Design</b>: Optimized for both desktop and mobile users.
# Technologies Used
Frontend: HTML, CSS
